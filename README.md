# 宠物领养管理系统

一个基于Java的宠物领养管理系统的实现。

## 功能特性
- 添加和管理宠物信息
- 添加和管理领养人信息
- 处理宠物领养流程
- 查看领养记录
- 搜索功能

## 项目结构
```
src/main/java/
├── com/
│   └── petadoption/
│       ├── entities/          # 实体类
│       ├── services/         # 服务接口
│       ├── impl/             # 服务实现
│       ├── managers/         # 管理类
│       └── Main.java         # 应用程序入口点
```
